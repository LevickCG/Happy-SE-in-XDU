

#算法分析与设计

## 简述

这门课以算法导论为课本对算法相关基本概念、经典算法思想、经典算法进行了系统论述。

四次实验，主题分别为：分治、动态规划、贪心、回溯。

作业：少量课后题。

另外邓凡老师会讲四次习题课，每次总结之前所学内容，并给出相应的补充问题。期末前进行考试题型讲解。



## 学习建议

学习需要构成框架，成体系，循序渐进。

实验宜自己动手，享受~~智商底下的无奈~~思维的乐趣。
